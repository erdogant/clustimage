SavingÔÉÅ

Saving and loading models can be used to restore previous results. In order to accomplish this, we created two functions: function clustimage.clustimage.Clustimage.save(). and function clustimage.clustimage.Clustimage.load(). Below we illustrate how to save and load models.

Saving the model with the results:

from clustimage import Clustimage

# Initialize
cl = Clustimage(method='hog')
# Load example data

X, y = cl.import_example(data='mnist')
# Preprocessing, feature extraction and cluster evaluation
results = cl.fit_transform(X)

# Load model
cl.save('clustimage_model', overwrite=True)

LoadingÔÉÅ

Loading a learned model can be done using the function clustimage.load():

from clustimage import Clustimage

# Load model
cl.load('clustimage_model')
